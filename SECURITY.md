# Security Policy

This is a small, single-maintainer project. The plugin talks to USB HID fan
controllers; it has no network surface and no server-side component.

## Reporting a Vulnerability

Please report security issues privately through GitHub's
[private vulnerability reporting](https://github.com/lewisgibson/FanControl.LianLi/security/advisories/new)
(the "Report a vulnerability" button on the Security tab) rather than opening a
public issue. Reports are handled on a best-effort basis. Please do not disclose
the issue publicly until a fix is available.

## Supported Versions

Only the latest release is supported. Older releases do not receive fixes.
